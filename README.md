# TranZak

TranZak is an African fintech platform headquartered in Douala, Cameroon, providing REST APIs for mobile money collections, merchant payments, bulk disbursements, and SMS services across sub-Saharan Africa. The platform bridges banked and unbanked populations by supporting MTN Mobile Money, Orange Money, and CEMAC bank networks.

## APIs

- **Payment Collections API** — Collect payments via mobile money, QR code, and web redirect checkout
- **Transfers (Disbursements) API** — Send funds to mobile wallets, bank accounts, and Tranzak users
- **Bulk Payments API** — Disburse to up to 2,000 recipients in a single request
- **SMS API** — Send marketing bulk SMS and OTP verification codes
